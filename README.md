# ✈️ Airline Reservation System

# Description

A Java-based application for managing airline reservations including flights and passengers.

##  Features

* Add and manage flights
* Book tickets
* Manage passengers
* Handle domestic & international flights

##  Technologies Used

* Java
* JavaFX

##  How to Run

1. Open the project in NetBeans or VS Code
2. Build the project
3. Run `AirLineReservationSystem.java`

##  Project Structure

* `src/` → Source code
* `build.xml` → Build file
